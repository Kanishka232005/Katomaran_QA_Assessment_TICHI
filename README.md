Defect Reports

Overview

This repository contains defect reports identified during the manual testing of the Tichi Web Application. The defects were documented based on my testing observations using a standard defect reporting format.

Each defect report includes:

- Defect ID
- Defect Title
- Defect Description
- Module
- Environment
- Preconditions
- Steps to Reproduce
- Expected Result
- Actual Result
- Severity
- Priority
- Test Data Used
- Screenshots
